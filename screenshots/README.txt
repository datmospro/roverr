# Screenshots for README

Please add the following screenshots to this folder:

## Required Screenshots:

1. **dashboard.png**
   - Take a screenshot of the main dashboard
   - Should show: Multiple movies with different statuses (New, Downloading, Seeding, Copied)
   - Make sure the UI looks clean and professional
   - Recommended resolution: 1920x1080 or 1600x900

2. **movie-details.png**
   - Take a screenshot of the movie details page
   - Click on any movie to open the details view
   - Should show: Movie poster, metadata, actions, status
   - Recommended resolution: 1920x1080 or 1600x900

3. **settings.png**
   - Take a screenshot of the Settings page
   - Navigate to Settings from the sidebar
   - Show either "General" or "Connection" tab (most important settings)
   - Recommended resolution: 1920x1080 or 1600x900

4. **search.png**
   - Take a screenshot of the Search interface
   - Navigate to Search from the sidebar
   - Show search results for a popular movie
   - Should display: Movie cards with posters and download options
   - Recommended resolution: 1920x1080 or 1600x900

## Optional (but recommended):

5. **demo.gif**
   - Short animated GIF showing the app in action
   - Suggestions:
     - Navigate through movies
     - View movie details
     - Perform a search
   - Tools: ScreenToGif (Windows), Kap (Mac), Peek (Linux)
   - Max size: 5MB
   - Duration: 10-20 seconds

## Tips:

- Use a clean browser window (hide bookmarks bar)
- Make sure no sensitive information is visible (passwords, API keys, etc.)
- Consider using dummy/example data if needed
- Optimize images before adding:
  - Use https://tinypng.com/ or https://squoosh.app/
  - Target: 200-500KB per image
  - Format: PNG for screenshots, GIF for animations

After adding the screenshots, they will automatically appear in the README.md!
